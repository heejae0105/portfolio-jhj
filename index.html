01. IoT 다인 탑승 감지 시스템
Arduino Sensor Fusion

전동 킥보드 2인 탑승이라는 사회적 안전 문제를 해결하기 위해, 다중 센서 융합 기술을 활용하여 실시간으로 위반 상황을 감지하고 경고하는 하드웨어 솔루션입니다.

개발 과정 및 상세 내용 보기 (Click)

1. 활동 목적 및 배경

전동 킥보드 2인 탑승은 국내에서 명백한 도로교통법 위반임에도 불구하고 일상에서 빈번하게 발생하는 안전 문제입니다. 단순한 법적 규제만으로는 실질적인 억제 효과를 기대하기 어렵다는 문제의식에서 출발하여, 킥보드 자체에서 2인 탑승을 감지하고 경고를 발생시키는 기술적 안전장치를 직접 설계·구현하였습니다.

2. 핵심 기술 구현

아두이노 보드를 중심으로 두 가지 센서를 복합 활용한 이중 감지 시스템을 설계했습니다.

  • FSR 압력 센서: 발판에 4개 배치, 3개 이상 동시 감지 시 조건 1 충족
  • 초음파 센서: 앞뒤 각 2개 배치, 탑승자 간 거리 합이 설정값 이하일 때 조건 2 충족
  • 경고 발생 로직: 두 조건이 동시에 충족되는 상황이 3회 이상 반복될 때만 부저 작동 → 일시적 오접촉을 필터링하고 실제 2인 탑승 상황에서만 경고 발생

3. 성과

센서 융합 기반의 조건 제어 로직을 직접 설계하고 구현하면서, 단일 센서의 오류 가능성을 보완하는 이중 검증 구조의 중요성을 체감했습니다. 나아가 교통 안전이라는 실질적인 사회 문제를 하드웨어와 소프트웨어의 결합으로 해결하는 과정을 경험하며, 기술이 사회에 기여할 수 있는 방향성에 대해 깊이 고민하는 계기가 되었습니다.

02. 익명 오픈채팅 — 탈중앙화 P2P 블록체인 메신저
Python Socket Blockchain

카카오톡 먹통 사례를 보고 중앙 서버에 대한 의존 없이 개인 간 직접 통신과 데이터 무결성을 동시에 보장하는 익명 오픈채팅 플랫폼 개발을 계획했습니다. Python Socket 기반의 P2P 네트워크 위에 블록체인 구조를 결합하여, 제3자의 개입 없이도 신뢰 가능한 통신 환경을 구현하였습니다.

개발 과정 및 상세 내용 보기 (Click)

1. 핵심 구현 사항

① 탈중앙화 P2P 네트워크
Python의 socket 라이브러리를 활용하여 각 클라이언트가 동시에 서버 역할을 수행하는 하이브리드 노드 구조를 설계하였습니다. 특정 노드가 연결을 종료하거나 장애가 발생하더라도 나머지 피어 간 통신이 중단 없이 유지되어 단일 장애점(SPOF) 문제를 구조적으로 해소하였습니다. 또한 중앙 데이터센터에 대한 의존도를 제거함으로써 로컬 환경에서도 독립적으로 운용 가능한 유연한 통신 인프라를 구축하였습니다.

② 블록체인 기반 메시지 무결성 보장
모든 대화 메시지는 단순 텍스트가 아닌, 인덱스·타임스탬프·데이터·이전 블록의 해시값으로 구성된 Block 객체로 생성됩니다. SHA-256 해시 함수를 통해 각 블록을 이전 블록과 수학적으로 연결함으로써 불변의 체인 구조를 형성합니다. 과거 메시지를 1바이트라도 수정할 경우 해당 블록의 해시값이 즉각 변경되어 이후 체인 전체가 무효화되므로, 중앙 서버의 보증 없이도 데이터 위·변조를 감지할 수 있습니다.

2. 기술적 의의

본 프로젝트는 가용성·비용 효율·정보 주권이라는 세 가지 과제를 단일 아키텍처로 해결하는 데 초점을 맞췄습니다. P2P 구조로 인프라 비용과 중앙화 리스크를 제거하고, 블록체인 알고리즘으로 신뢰 기관 없는 무결성 검증을 실현하여 완전한 탈중앙화 메신저 환경을 구현하였습니다.

03. DealChecker: AI 기반 계약서 분석 웹 (ACT 해커톤)
AI / NLP Fine-Tuning Flask

청년들의 근로계약서 피해 예방을 위해, 사용자가 쉽게 불공정 조항을 인식하고 점검할 수 있도록 AI를 파인튜닝하여 시각화해 주는 법률 지원 웹 서비스입니다.

개발 과정 및 상세 내용 보기 (Click)

1. 개발 동기 및 목적

진행 개요: ACT 해커톤 출품작 (2025.07.31 ~ 08.02, 대전대신고등학교)

대전 지역 청년들이 근로계약서 미작성이나 불공정 조항으로 겪는 피해를 예방하기 위해 기획했습니다. 정보 불균형과 법률 서비스 접근성 문제를 해결하고자, 누구나 활용 가능한 사회 문제 해결형 웹사이트를 개발하여 지역 사회 청소년이 참여하는 IT 공헌 모델을 제시하고자 했습니다.

2. 적용 기술 및 이론적 배경

  • 자연어 처리 (NLP): 사용자가 업로드한 계약서 문장을 단어 단위로 분리하고 의미를 파악하여, 불공정 조항 여부를 자동으로 탐지하고 분류하는 데 활용했습니다.
  • AI 파인튜닝 (Fine-Tuning): 사전 학습된 범용 인공지능 모델에 실제 근로계약서 사례와 불공정 조항 데이터셋을 추가로 학습시켜, 법률 문서라는 특정 도메인에 최적화된 높은 정확도의 분석 결과를 도출했습니다.

3. 개발 과정

  • 데이터 전처리: 실제 계약서 사례를 조사하여 불공정 조항의 패턴 문장을 발췌하고, 모델 학습에 적합한 형태로 데이터셋을 가공했습니다.
  • 모델 학습 및 검증: 파인튜닝을 통해 정확도와 재현율을 평가·개선하고, 가상의 계약서 입력 및 법률 사료 비교를 통해 신뢰도를 검증했습니다.
  • 웹사이트 구현: Python(Flask) 기반 백엔드와 HTML/CSS를 활용해, 사용자가 계약서를 업로드하면 자동으로 분석 결과를 시각화해 주는 인터페이스를 구축했습니다.

4. 주요 성과 및 결과물

법률 지식이 부족한 사용자도 스스로 계약 내용을 점검할 수 있는 직관적인 플랫폼을 완성했습니다. 특히 플랫폼 내에 전용 관리 페이지를 개발하여, 사용자가 과거에 업로드한 계약서와 분석 결과를 누적하여 저장, 비교, 검토할 수 있는 체계적인 이력 관리 기능을 구현했습니다.
