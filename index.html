01. IoT 다인 탑승 감지 시스템
Arduino C++ Sensor Fusion

단순 무게 감지 방식의 오작동 한계를 극복하기 위해, 다중 센서를 융합(Sensor Fusion)하여 2인 탑승 오탐률을 획기적으로 줄인 하드웨어 기반 안전 솔루션입니다.

개발 과정 및 상세 내용 보기 (Click)

1. 문제 정의 (Why?)

기존 전동 킥보드는 단순 '무게'만으로 탑승자를 감지하여, 무거운 짐을 실었을 때도 경고가 울리는 오작동이 빈번했습니다. '사람 2명'의 특징은 무게뿐만 아니라 '발판을 차지하는 면적'과 '앞뒤 공간의 협소함'이라는 물리적/공간적 특징이 있다는 것에 착안했습니다.

2. 핵심 기술 구현 (How?)

단일 센서의 한계를 극복하는 Sensor Fusion (센서 융합) 알고리즘 설계

  • 물리적 조건 (FSR 압력 센서): 발판의 4지점 중 3곳 이상에서 압력이 감지되는지 분석하여 사람의 발 위치를 파악합니다.
  • 공간적 조건 (초음파 센서): 탑승자의 전/후방 거리가 기준치 이하로 밀집되었는지 교차 검증합니다.

3. 트러블 슈팅 & 성과

초기에는 센서 값이 순간적으로 튀는 '노이즈' 현상 때문에 경고음이 오작동했습니다. 이를 해결하기 위해 두 조건이 동시에 3회 이상(Cycle) 연속으로 감지될 때만 '위험'으로 판단하는 소프트웨어적 필터링(Debouncing) 로직을 추가했습니다. 현실 세계의 Raw Data를 다루고 보정하는 실무적인 감각을 익힐 수 있었습니다.

02. P2P 탈중앙화 블록체인 메신저
Python Socket Blockchain

카카오톡 먹통 사태에서 착안하여, 중앙 서버에 의존하지 않고 클라이언트 간 직접 통신하며 데이터 위변조를 차단하는 메신저를 네트워크 밑바닥부터 직접 구현했습니다.

개발 과정 및 상세 내용 보기 (Click)

1. 탐구 배경

대부분의 메신저는 중앙 서버(SPOF)가 다운되면 모든 소통이 마비됩니다. 완성된 통신 API를 쓰기보다, 네트워크의 기본인 TCP/IP 통신의 원리를 깊이 이해하고자 직접 소켓 프로그래밍에 도전했습니다.

2. 핵심 기술 구현

  • Socket Programming 기반 P2P: 서버를 거치지 않고 Client A와 Client B가 IP와 Port를 통해 직접 패킷을 주고받는 탈중앙화 통신 환경을 구축했습니다.
  • 블록체인 해시 무결성 검증 (Data Integrity): 해킹 및 데이터 변조를 막기 위해 블록체인의 Chain 구조를 도입했습니다. 이전 메시지의 해시(Hash)값을 다음 메시지 전송 시 꼬리표처럼 붙여, 누군가 대화를 몰래 수정(MITM 공격)하면 해시값이 어긋나 즉시 경고를 띄우도록 설계했습니다.

3. 배운 점

편리한 라이브러리에 가려져 있던 네트워크 계층의 데이터 흐름과 암호화의 원리를 직접 코드로 짜보며 탄탄한 백엔드 지식의 기반을 마련했습니다.

03. DealChecker: AI 계약서 분석 웹
AI / NLP Fine-Tuning Full-Stack

어려운 법률 용어로 인한 계약 피해를 예방하고자, AI를 파인튜닝하여 계약서 내 불공정/독소 조항을 자동으로 찾아내고 시각화해 주는 웹 서비스를 개발했습니다.

개발 과정 및 상세 내용 보기 (Click)

1. 프로젝트 개요

"나의 코딩이 어떻게 사회적 가치를 만들 수 있을까?"에 대한 답을 찾고자 기획했습니다. 일반인이 놓치기 쉬운 불공정 약관을 자연어 처리(NLP) 기술을 활용해 분석합니다.

2. 기술적 챌린지 (AI 파인튜닝)

단순한 프롬프트 엔지니어링만으로는 범용 AI가 법률 전문 용어와 맥락을 정확히 짚어내지 못했습니다. 이를 해결하기 위해 '불공정 약관 데이터셋'을 직접 수집 및 전처리하여 언어 모델을 파인튜닝(Fine-Tuning)했습니다. 입력(계약서 원문)과 출력(위험 조항 도출) 데이터를 일관되게 정제하는 과정에서 AI 학습 파이프라인 전반을 경험했습니다.

3. 의의 및 성과

백엔드 서버에서 AI 모델을 서빙하고, 프론트엔드 웹 화면에서 위험 조항을 붉은색으로 하이라이팅 처리하여 사용자에게 보여주는 전체 서비스 흐름(Full-Stack)을 직접 설계하고 구현해 낸 뜻깊은 프로젝트입니다.
